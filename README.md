# Leela - A basic Smoothed Particle Hydrodynamics solver

## Running Leela
To compile Leela, make sure you are in the source directory and use the makefile.
```
cd leela
make
```
This will build Leela in the `bin` directory. To run leela:
```
cd bin
./leela
```
